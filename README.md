# 💔 Emotional Speech Website - "I'm Sorry"

A beautiful, responsive website featuring an emotional apology speech with "Tum Hi Ho" music background.

## 🎨 Features

- **Dark Emotional Theme** with gold and red accents
- **Background Music** - "Tum Hi Ho" (Aashiqui 2)
- **Smooth Animations** - Fade-ins, slide-ups, scroll effects
- **Interactive Elements** - Music player, custom buttons
- **Floating Hearts** - Double-click anywhere for floating heart animations
- **Responsive Design** - Works on all devices
- **Progress Bar** - Visual scroll indicator

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Interactive functionality
- `README.md` - Documentation (this file)

## 🚀 How to Use

1. **Download/Clone** all files to a folder
2. **Open `index.html`** in your web browser
3. **Click the 🎵 button** (bottom-right) to play/pause music
4. **Scroll** to see animations
5. **Double-click** anywhere for floating hearts
6. **Click "Send My Feelings"** button for heart animation
